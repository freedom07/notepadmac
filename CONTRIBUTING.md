# Contributing to NotepadNext

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/notepadnext.git`
3. Create a feature branch: `git checkout -b feature/my-feature`
4. Build: `swift build`
5. Test: `swift test`

## Development

- **macOS 13+** and **Swift 5.9+** required
- Follow existing code style (SwiftFormat/SwiftLint configured)
- Add tests for new features
- Keep commits atomic and descriptive

## Pull Requests

- Open a PR against `main`
- Describe what your PR does and why
- Ensure `swift build` and `swift test` pass
- One feature per PR

## Reporting Issues

- Use the issue templates (bug report / feature request)
- Include macOS version and reproduction steps for bugs

## Code of Conduct

Be respectful and constructive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
