# Changelog

### Added
- Piece Table text engine with undo/redo
- Syntax highlighting for 19 languages
- 16 built-in color themes
- Tab interface, line numbers, minimap, split view
- Find/replace with regex, multi-cursor, column editing
- Command palette, macro system, plugin architecture
- Markdown preview, code folding, bracket matching
- File encoding detection, auto-save, session restore
